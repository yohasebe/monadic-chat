# Monadic Chat ドキュメント

Monadic Chat ドキュメントへようこそ！

Monadic Chatは、インテリジェントなチャットボットを作成・利用するためのローカル環境で動作するWebアプリケーションです。Docker上の実際のLinux環境をAIモデルに提供することで、外部ツールを必要とする高度なタスクを実行できます。このドキュメントでは、Monadic Chatのインストール方法、使用方法、拡張方法について説明します。
