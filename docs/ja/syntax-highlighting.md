# シンタックス・ハイライト

Monaco Editor は、シンタックス・ハイライトをサポートしています。シンタックス・ハイライトは、コードの構文を強調表示するための色付けです。Monaco Editor は、シンタックス・ハイライトを有効にするために、`monaco.languages.register` メソッドを使用します。シンタックス・ハイライトのテーマは設定パネルで選択できます。デフォルトは `monokai` です。
