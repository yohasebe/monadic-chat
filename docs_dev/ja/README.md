# Monadic Chat 内部ドキュメント

## 概要

これはMonadic Chatの内部開発者向けドキュメントです。実装、アーキテクチャ、開発ワークフローに関する技術的な詳細が含まれており、エンドユーザーには関係のない内容です。

**ユーザー向けドキュメント**については、[公開ドキュメント](https://yohasebe.github.io/monadic-chat/)をご覧ください。

## ドキュメントガイドライン

### 内部 vs 外部ドキュメント

**内部ドキュメント（`docs_dev/`）に含めるもの：**
- 実装の詳細とアーキテクチャの決定事項
- 内部APIとデータフロー
- 開発ワークフローとデバッグ
- テスト手順とインフラストラクチャ
- 技術的負債と改善計画

**外部ドキュメント（`docs/`）に含めるもの：**
- ユーザー向けの機能と機能性
- インストールとセットアップ手順
- 基本的な使用ガイドとチュートリアル
- 公開APIドキュメント
- ユーザー向けトラブルシューティングガイド

### ドキュメントを最新に保つ

Betaピリオド（1.0以前）では、以下の原則に従います：
- **古い情報は掲載しない** - 常に現在の情報で置き換える
- **積極的なメンテナンス** - コードを変更したらドキュメントも更新する
- **単一の情報源** - 各トピックには1つの権威あるドキュメントのみ

## ローカルドキュメントへのアクセス

デバッグモード（`rake server:debug`）で実行している場合、内部・外部両方のドキュメントがローカルで利用できます：
- 内部ドキュメント: http://localhost:4567/docs_dev/
- 外部ドキュメント: http://localhost:4567/docs/

詳細は**[デバッグモードとローカルドキュメント](server-debug-mode.md)**を参照してください。
