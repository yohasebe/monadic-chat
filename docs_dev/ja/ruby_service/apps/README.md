## 概要

Monadic Chatの各アプリケーションはMDSL（Monadic DSL）を使用して定義されており、特定の実装詳細をここに文書化できます。これらのドキュメントは以下をカバーします：

- アプリ固有のアーキテクチャ決定
- ツールメソッドの実装
- AIプロバイダーとの統合パターン
- 特別な機能や能力
