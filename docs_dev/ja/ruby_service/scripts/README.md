# Rubyサービススクリプトドキュメント

このディレクトリには、Rubyサービスの開発とメンテナンスで使用されるユーティリティスクリプトのドキュメントが含まれています。

## コンテンツ

- [概要](overview.md) - 利用可能なスクリプトとその目的の一般的な概要
- [診断](diagnostics/) - 診断とトラブルシューティングスクリプト

## スクリプトカテゴリ

### 診断
トラブルシューティングとシステムヘルスチェック用のスクリプト。特定のツールについては`diagnostics/`サブディレクトリを参照してください。

### 開発ツール
コード生成、データベース管理、サービスオーケストレーションヘルパーなど、開発ワークフローをサポートするユーティリティスクリプト。
